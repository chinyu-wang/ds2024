數位敘事實作與應用

本課程主要在介紹數位敘事相關技術與應用，
包含網頁設計、地圖設計、圖表設計等，
利用易用及有用的電腦軟硬體工具，將資訊內容以視覺多元化的方式呈現出來。

繼續...

數位敘事六步驟:

1.定義主題
2.進行研究
3.擬定文稿
4.準備素材 (Image、Video、Audio...)
5.整合素材
6.分享作品

課程主要內容

包含網頁敘事、圖表敘事、地圖敘事、新媒體敘事等

網頁敘事

介紹HTML、CSS、Javascript 原理、靜態網頁實作、Open Data 開放資料搜集與應用。

圖表與地圖敘事

利用多種網頁元件繪製地圖與圖表。

新媒體敘事

以 Virtual Space 製作為主。

工具與技術應用

必備工具 - Sublime Text, Mobirise, Github
基礎技術 - HTML, CSS, Javascript
 地圖工具 - Google My Map
圖表工具 - RAW Graphs, High charts, amCharts
新媒體工具 - GOXR, Knight Lab Projects

其他參考工具 (optional)
虛擬實境 - KRPANO
互動簡報 - Prezi
動畫影片 - Powtoon
心智圖 - XMind: Zen
圖像 - Adobe Spark
資訊圖表 - Infogram
遊戲 - Google 簡報
懶人包 - PiktoChart
APP - App Inventor 2

評量方式

一、期中報告 (30%)
二、平時作業及課程參與 (40%)
三、期末報告 (30%)

曠課每次扣總成績3分，情節嚴重者加重扣分。

平時作業及課程參與

包含出缺席、課堂練習、自主學習等面向。

[16+2自主學習]

1.參觀相關策展並上傳心得報告
2.閱讀相關文章並上傳心得報告

以上擇一
心得報告請包含影像1張及300字以上內容
